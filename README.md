ASP.NET MVC 購物網功能實作
=======================
> ## 說明:
這是下班之餘跟著網路資源自學ASP.NET MVC所做的購物網站功能實作

> ## 使用技術:
1.後端 :
* .Net Framework MVC框架
* Entity Framework 這裡使用DB first，使用MSSQL的LocalDB
* Web API 提供前端使用
2.前端 :
* HTML/CSS/JS
* bootstrap
* AJAX介接Web API更新購物車內容
